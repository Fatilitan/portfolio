Prisma API

For my final assignment, I had to set up a relational database using Prisma and develop an API with Express.js. The project simulates the backend of a vacation-rental platform similar to Airbnb. Thus, the relational models included properties, hosts, users, reviews, etc. After initializing and seeding the database with dummy data, I proceeded to write HTTP requests for retrieving, updating, posting, and deleting data in the database. These requests are designed to handle errors appropriately and are implemented with authentication for specific endpoints.
