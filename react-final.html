React Assignment

For the React assignment, I was tasked with developing a dashboard to manage events. The dashboard needed to perform several functions: creating, deleting, updating, and retrieving events; searching by event name and/or filtering by event category; and navigating to detailed pages about specific events. To facilitate navigation, I utilized React Router. Additionally, I employed React Router to handle operations such as retrieving, creating, deleting, and updating events. The events were stored locally on a "json-server" named events.json.
