IP Adress Tracker

The IP Address Tracker is a website designed to track any IP address or domain. The website is built using HTML, CSS, and JavaScript. It incorporates two APIs: ipify and Leaflet.js. ipify is used for searching domains and IP addresses, while Leaflet.js is utilized to display the location of the searched IP address or domain.
